# Twitter-Like-React-Component

This project aims to make 

**Easily Configurable**

**Customizable**

**Responsive**

**CSS Free**

react component that looks like Twitter.


## Currently

Working on component implementation.


You can test this library's alpha version by using

```bash
npm install tlrc
```

or

```bash
yarn add tlrc
```
